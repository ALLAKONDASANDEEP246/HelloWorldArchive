# Python Hello World

## 1. How to Install
- **Python 3**: Download from [python.org](https://www.python.org/downloads/).
- Verify installation:  
```bash
python --version  # Should show Python 3.x
```

## 2. How to Run  
For Python 3:
```sh
python hello-world.py
```

## 3. Output
```bash
Hello, World!
```
